
# COVID-19 Data Analysis Report

## Objective
Analyze global and country-level COVID-19 data to understand case trends and comparisons.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
https://www.kaggle.com/datasets/imdevskp/corona-virus-report

## Features
- Daily trend analysis
- Rolling average trends
- Country-wise comparison
